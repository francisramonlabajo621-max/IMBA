Nexus Arena is a Flask-powered blog inspired by the structure of `FinsFriend`, rebuilt for esports news, strategy breakdowns, and community commentary.

## Features

- Post library with search and category filters (MOBA, FPS, Tactics, etc.)
- Post CRUD for authenticated users
- Comment threads tied to each post
- Login/logout/register flows using `Flask-Login`
- SQLite persistence with auto-seeded demo content
- Responsive neon-inspired UI built with a single CSS file

## Getting started

```bash
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
set FLASK_APP=app.py
python app.py
```

Visit [http://localhost:5010](http://localhost:5010) to explore the demo posts.

## Reference

This project mirrors the layout and architecture of `C:\Users\Stephen\FinsFriend` while substituting esports-specific copy, imagery, and palettes.
